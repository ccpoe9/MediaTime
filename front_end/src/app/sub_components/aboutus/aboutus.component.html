ABOUT US :

- My name is Christian Okey-Ezeh.
- I chose this project because i've always wondered what went into building a video streaming site.
- The techonologies used were what best fit my needs. Angular for front-end with firebase for authentication, Node.js because of it's lightweight structure and simplicity and mysql for the database.
- Also made use of the videojs javascript package for video players due to it's support for both live and vod content.
- Stored all video content in wasabi storage, while the live streams were pulled from a iptv.org github page (All legal).
- Deployed backend and database in a dockerize container to digital ocean and the front end to netlify.
